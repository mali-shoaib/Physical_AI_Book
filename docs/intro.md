# Physical AI & Humanoid Robotics

Welcome to the **AI-Native Textbook** for Physical AI and Humanoid Robotics!

## About This Textbook

This comprehensive textbook is designed for robotics students, developers, and researchers who want to master the integration of artificial intelligence with physical robotics systems, with a focus on humanoid robots.

### What You'll Learn

- **Module 1**: The Robotic Nervous System (ROS 2) - Master ROS 2 middleware, nodes, topics, services, and URDF for robot control
- **Module 2**: Simulation Environments - Build and test robots in Gazebo and Unity
- **Module 3**: NVIDIA Isaac Sim - Leverage GPU-accelerated physics and synthetic data generation
- **Module 4**: Vision-Language-Action Models - Integrate cutting-edge AI models with robot control
- **Module 5**: Capstone Project - Build an end-to-end humanoid robot system

## Prerequisites

To get the most out of this textbook, you should have:

- **Programming**: Basic Python knowledge (functions, classes, imports)
- **Linux**: Familiarity with terminal commands (cd, ls, source)
- **Mathematics**: Understanding of basic linear algebra and trigonometry
- **Optional**: Previous robotics experience helpful but not required

## Setup

### Software Requirements

- **ROS 2 Humble** (Long-Term Support release)
- **Python 3.10+**
- **Ubuntu 22.04 LTS** or Docker container

### Installation

Follow the official ROS 2 Humble installation guide:
[https://docs.ros.org/en/humble/Installation.html](https://docs.ros.org/en/humble/Installation.html)

Alternatively, use Docker:
```bash
docker pull osrf/ros:humble-desktop
```

## How to Use This Textbook

### For Students

1. **Follow sequentially** - Chapters build on previous knowledge
2. **Run all examples** - Hands-on practice is essential
3. **Use the chatbot** - Ask questions as you learn
4. **Complete exercises** - Reinforce your understanding

### For Instructors

This textbook can be used for:
- University robotics courses (undergraduate/graduate)
- Professional training workshops
- Self-paced online learning

Each module is designed to be completable in 2-4 weeks of study.
