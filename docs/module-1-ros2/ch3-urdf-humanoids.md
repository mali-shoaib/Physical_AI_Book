## Learning Objectives

By the end of this chapter, you will be able to:

- Understand URDF (Unified Robot Description Format) syntax
- Create humanoid robot models with joints and links
- Add sensors (cameras, IMUs, LiDAR) to robot models
- Use Xacro for modular robot descriptions
- Visualize robots in RViz2

## Prerequisites

- Completed [Chapter 2: rclpy Control](/docs/module-1-ros2/ch2-rclpy-control)
- Basic understanding of 3D coordinate systems
- Familiarity with XML syntax
